# 🔒 Nael Security Scanner

Professional Website Security Audit Tool

## ✨ Features

- 🔐 **Security Headers** - HSTS, CSP, X-Frame-Options, etc.
- 🔑 **SSL/TLS** - Certificate validation, TLS version
- 🌐 **DNS/WHOIS** - DNS records, domain expiration
- 📄 **HTML Analysis** - Metadata, links, images, validation
- ⚡ **Performance** - Load time, page size, resources
- 📡 **Network** - SSID, IP, latency, signal strength
- 📊 **Reports** - JSON & HTML with detailed analysis

```bash
# 1. Clone
git clone https://github.com/nathangabrilganteng-arch/Nael.git
